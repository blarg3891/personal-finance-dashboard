# Beeron Tech Dashboard

![Beeron Tech Logo](logo.png)

## About
Beeron Tech Dashboard is a personal wealth management application designed to aggregate investment data across multiple brokerage accounts. It utilizes the SnapTrade API to provide a unified view of net worth, asset allocation, and performance metrics.

## Features
* **Automated Data Sync:** Secure connection to brokerage accounts (Fidelity, Webull, etc.) via SnapTrade.
* **Portfolio Analysis:** Real-time calculation of asset distribution and exposure.
* **Privacy First:** Data is processed locally for personal use only.
